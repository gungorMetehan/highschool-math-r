# Lise Matematik - R Uygulamaları

Bu repository'nin amacı Türkiye'de liselerde (ortaöğretim) okutulan matematik kitaplarındaki konuları destekleyecek şekilde kullanılabilecek R komut dosyaları oluşturmaktır. Komut dosyaları hazırlanırken Türkiye Yüzyılı Maarif Modeli kapsamında Millî Eğitim Bakanlığı'nın [resmi sitesinde](https://tymm.meb.gov.tr/ders-kitaplari/ortaogretim) yayımlanan kitaplardaki temalar / konular temel alınmıştır.

## 9. Sınıf
1. Tema: Sayılar
2. Tema: Nicelikler ve Değişimler
3. Tema: Geometrik Şekiller
4. Tema: Eşlik ve Benzerlik
5. Tema: Algoritma ve Bilişim
6. Tema: İstatistiksel Araştırma Süreci
7. Tema: Veriden Olasılığa
